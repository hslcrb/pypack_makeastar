# makeastar

별 찍기 패턴을 출력하기 위한 **초경량** 파이썬 패키지입니다.  
복잡한 반복문 없이, 임포트하고 호출하기만 하면 됩니다.

## 특징

- **최적화**: 제너레이터 표현식과 f-string을 사용하여 효율성을 극대화했습니다.
- **쉬운 사용**: 직관적인 함수 이름과 **한국어 발음 별칭** 지원 (예: `star.samgak(5)`).
- **유연성**: 삼각형의 가로와 세로 길이를 자유롭게 조절 가능.
- **경량화**: 코드가 매우 간결합니다.

## 설치

```bash
pip install makeastar
```

(또는 로컬 설치)

```bash
pip install .
```

## 사용법

```python
import star
# 또는 한국어 패키지명 사용 가능
import 별

# 1. 삼각형 (별칭: 삼, ㅅㄱ, 삼각형)
별.삼(5)
별.삼각형(5)
star.ㅅㄱ(5)

# 2. 우측 정렬 삼각형 (별칭: 우삼, ㅇㅅㄱ, 우측삼각형)
별.우삼(5)

# 3. 역삼각형 (별칭: 역삼, ㅇㅅ, 역삼각형)
별.역삼(5)
star.ㅇㅅ(5)

# 4. 우측 정렬 역삼각형 (별칭: 우역, ㅇㅇ, 우측역삼각형)
별.우역(5)

# 5. 피라미드 (별칭: 피라, ㅍㄹ, ㅍㄹㅁㄷ, 피라미드)
별.피라(5)
별.피라미드(5)
star.ㅍㄹㅁㄷ(5)

# 6. 다이아몬드 (별칭: 다, ㄷㅇ, 다이아몬드)
별.다(5)

# 7. 모래시계 (별칭: 모, ㅁㄹ, 모래시계)
별.모(5)

# 8. 화살표 (별칭: 화, ㅎㅅ, 화살표)
별.화(5)
```
