# makeastar 기여 가이드

`makeastar`에 관심을 가져주셔서 감사합니다! 우리는 모든 분들의 기여를 환영합니다.

## 기여 방법

1. 저장소를 자신의 GitHub 계정으로 **포크(Fork)**합니다.
2. 프로젝트를 로컬 머신에 **클론(Clone)**합니다.
3. 변경 사항을 잘 설명하는 간결한 이름으로 **브랜치(Branch)**를 생성합니다.
4. 변경 사항을 **커밋(Commit)**합니다.
5. 포크한 저장소에 **푸시(Push)**합니다.
6. 원본 저장소에 **풀 리퀘스트(Pull Request)**를 보냅니다.

## 코드 스타일

- 모든 코드 주석과 문서는 **영어**로 작성해 주세요.
- 코드는 최소한으로 유지하고 최적화해 주세요. 장황한 루프보다는 한 줄 코드(one-liners)나 고효율 제너레이터 표현식을 선호합니다.
- 가능한 한 PEP 8 스타일 가이드를 따라 주세요.

## 버그 제보

버그를 발견하시면 다음 내용을 포함하여 이슈를 생성해 주세요:
- 무엇을 했는지.
- 어떤 결과가 예상되었는지.
- 실제로 어떤 결과가 발생했는지.
- 사용 환경 (OS, Python 버전).

`makeastar`를 더 좋게 만드는 데 도움을 주셔서 감사합니다!
